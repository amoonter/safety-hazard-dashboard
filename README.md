# Safety Hazard Dashboard

公司安全隐患分析仪表盘静态发布版。

## 访问方式

GitHub Pages 启用后，访问：

https://amoonter.github.io/safety-hazard-dashboard/

## 文件说明

- `index.html`：页面入口
- `app.js`：数据解析、筛选、统计和图表渲染逻辑
- `styles.css`：页面样式
- `data/2026-hazards-cleaned.json`：当前发布版使用的清洗后安全隐患数据

## 数据更新

替换 `data/2026-hazards-cleaned.json` 后重新提交并推送即可。
